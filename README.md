# Projeto 1 - Parte 1 - FRC

Repositório para hospedar o código da primeira parte do primeiro projeto da matéria Fundamentos de Redes de Computadores (FRC) na Universidade de Brasília (UnB).

Para funcionamento dos códigos, eles devem ser executados de forma paralela, isto é, os 3 ao mesmo tempo, em 3 terminais/processos diferentes. Além disso, é importante criar uma pasta 'arquivo-criado' para que o arquivo novo seja guardado.
